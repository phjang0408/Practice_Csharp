# C# 학습 로드맵 (Unity 목표)

---

## 1단계: 언어 기초

### 1-1. 환경 설정 ✅
- .NET SDK 설치, Visual Studio Community / VS Code + C# Dev Kit
- 콘솔 앱으로 Hello World 실행

### 1-2. 기본 타입 & 변수 ✅
- 값 타입: `int`, `float`, `double`, `bool`, `char`
- 참조 타입: `string`, `object`
- `var` 키워드, 암시적 형변환 vs 명시적 형변환
- `const`, `readonly`

### 1-3. 연산자 & 제어 흐름 ✅
- 산술, 비교, 논리 연산자
- `if/else if/else`, 삼항 연산자
- `switch` (expression switch 포함)
- `for`, `while`, `do-while`, `foreach`
- `break`, `continue`

### 1-4. 메서드
- 매개변수, 반환값, `void`
- `ref`, `out` 매개변수
- 선택적 매개변수, 명명된 인수
- 메서드 오버로딩

### 1-5. 배열 & 컬렉션
- 1차원/다차원 배열
- `List<T>`, `Dictionary<K,V>`, `HashSet<T>`, `Queue<T>`, `Stack<T>`
- 컬렉션 순회

---

## 2단계: 객체지향 (OOP)

### 2-1. 클래스 & 구조체
- 클래스 선언, 인스턴스 생성 (`new`)
- 필드, 프로퍼티 (`get`/`set`, auto-property)
- 생성자, 생성자 오버로딩
- `struct` vs `class` 차이 (값 타입 vs 참조 타입)

### 2-2. 접근 제한자
- `public`, `private`, `protected`, `internal`
- 캡슐화 원칙

### 2-3. 상속
- `: BaseClass` 문법
- `base` 키워드
- `virtual` / `override` / `sealed`
- 생성자 체이닝

### 2-4. 추상 클래스 & 인터페이스
- `abstract class`, `abstract` 메서드
- `interface` 정의 & 구현
- 다중 인터페이스 구현
- 추상 클래스 vs 인터페이스 차이

### 2-5. 다형성
- 업캐스팅 / 다운캐스팅
- `is`, `as` 연산자
- 런타임 타입 확인

### 2-6. 정적 멤버 & 유틸리티
- `static` 필드, 메서드, 클래스
- `this` 키워드
- `partial` 클래스

### 2-7. 예외 처리
- `try / catch / finally`
- 예외 계층 구조 (`Exception`, `ArgumentException` 등)
- 커스텀 예외 클래스 작성
- `throw` vs `throw ex`

## 3단계: 중급

### 3-1. 제네릭
- 제네릭 클래스, 메서드
- 타입 제약 (`where T : class`, `where T : new()` 등)
- 공변성 / 반공변성 기초

### 3-2. 델리게이트 & 이벤트
- `delegate` 선언 & 호출
- `Action<>`, `Func<>`, `Predicate<>`
- 멀티캐스트 델리게이트
- `event` 키워드, 발행-구독 패턴

### 3-3. 람다 & 익명 메서드
- `=>` 문법
- 클로저 & 변수 캡처
- 식 본문 멤버 (`=>` in properties/methods)

### 3-4. LINQ
- 메서드 체이닝: `Where`, `Select`, `OrderBy`, `GroupBy`, `First`, `Any`, `All`, `Count`
- 쿼리 문법 (`from ... where ... select`)
- `IEnumerable<T>` vs `IQueryable<T>`
- 지연 실행 (deferred execution)

### 3-5. 널 처리
- `null` 처리, `NullReferenceException` 방지
- nullable 타입: `int?`, `string?`
- null 병합 연산자: `??`, `??=`
- null 조건 연산자: `?.`, `?[]`

### 3-6. 비동기 프로그래밍
- 동기 vs 비동기 개념
- `Task`, `Task<T>`
- `async` / `await`
- `ConfigureAwait`, 취소 토큰 (`CancellationToken`)

### 3-7. 모던 C# 문법
- 패턴 매칭: `switch` expression, `is` pattern
- `record` 타입
- 튜플 & 분해 (`(a, b) = ...`)
- `using` 선언, 범위 지정 `using`

## 4단계: Unity

### 4-1. Unity 기초
- `MonoBehaviour` 생명주기: `Awake → OnEnable → Start → Update → OnDisable → OnDestroy`
- `GameObject`, `Component` 구조
- `Transform`, `Vector2/3`, `Quaternion`

### 4-2. Unity C# 패턴
- `[SerializeField]`, `[Header]` 등 어트리뷰트
- `GetComponent<T>()`, `FindObjectOfType<T>()`
- `Instantiate`, `Destroy`
- Coroutine (`IEnumerator`, `StartCoroutine`)

### 4-3. 물리 & 입력
- `Rigidbody`, `Collider`, 충돌 이벤트
- `Input` 시스템 (Legacy & New Input System)

### 4-4. 씬 & 에셋 관리
- `SceneManager`
- `Resources.Load`, Addressables 기초
- ScriptableObject

### 4-5. 실전 프로젝트
- 간단한 2D 플랫포머 또는 탑다운 슈터 완성
- UI 연동 (`Canvas`, `Button`, `TextMeshPro`)
- 빌드 & 배포
